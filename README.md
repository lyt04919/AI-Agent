# AI 多智能体智能客服系统 MVP (完整项目包)

本项目它展示了如何使用多智能体协作（Multi-Agent Collaboration）来解决复杂的企业级客诉问题。

## 目录结构
- `app.py`: 基于 Streamlit 的可视化交互界面及 Agent 核心逻辑。
- `requirements.txt`: 项目所需的 Python 依赖库。
- `.env.example`: 环境变量模板。
- `application_guide.txt`: 文字材料（痛点 + 逻辑流描述）。

## 快速启动
1. 安装依赖: `pip install -r requirements.txt`
2. 配置环境: 将 `.env.example` 改名为 `.env` 并填写 API Key。
3. 启动应用: `streamlit run app.py`
